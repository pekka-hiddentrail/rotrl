## Requirements
- **Response Format**: Plain text only
- **Expected Output**: `hello world`
- **No additional text**: Do NOT add explanations, punctuation, or any other content
- **No formatting**: Do NOT use markdown, code blocks, or special formatting
- **Single response**: Provide exactly one line containing only: `hello world`

## Strict Response Pattern
```
hello world
```

That is the ONLY acceptable response. No more, no less.

Your task: Say hello to the world.

## Response
