# Delta Template — copy to session_NNN.md inside an NPC folder

Filename convention: session_001.md, session_002.md, etc.
These files are git-ignored — they record what happened with this NPC during that session.

---

# Session NNN Delta — YYYY-MM-DD

## What Happened
- Brief bullet points of significant interactions this session
- Any trust changes, revelations, or decisions made

## Current State
**Disposition:** Cooperative / Neutral / Distrustful / Hostile
**Location:** Where they are now (if changed from base)
**Trust Level:** Rough description — e.g. "trusts the party after goblin defence"

## Open Threads
- Any promises made, debts owed, or unresolved tensions

## Notes
Free-form GM notes about tone, what the NPC knows, what not to reveal yet.
