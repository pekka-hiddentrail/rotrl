# GM Operating Rules (Index)

The GM Operating Rules have been split into three files by priority and frequency of use:

1. [GM_OPERATING_RULES_01_CRITICAL.md](GM_OPERATING_RULES_01_CRITICAL.md)  
	Highest-priority rules for session start and first interactions.

2. [GM_OPERATING_RULES_02_GUIDELINES.md](GM_OPERATING_RULES_02_GUIDELINES.md)  
	Ongoing operational guidance for active play.

3. [GM_OPERATING_RULES_03_TRIVIAL.md](GM_OPERATING_RULES_03_TRIVIAL.md)  
	Lower-frequency edge cases and administrative constraints.

All three files together define the authoritative GM operating behavior.
