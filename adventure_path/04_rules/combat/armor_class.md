# Armor Class

**Triggers:** armor class, ac, touch ac, flat-footed, cmd, combat maneuver defense, what's my ac, what is ac, ac is, defense, defended by

## Resolution

`AC = 10 + armor + shield + DEX + size + natural + deflect + misc`

| AC Type | What it stops |
|---|---|
| Full AC | Normal melee and ranged attacks |
| Touch AC | Spells/abilities that require a touch (ignore armor + shield + natural) |
| Flat-Footed AC | Attacker before initiative or attacker with Sneak Attack (loses DEX bonus, unless Uncanny Dodge) |

## CMD (Combat Maneuver Defense)

`CMD = 10 + BAB + STR mod + DEX mod + size mod + misc`

Used as the DC for bull rush, disarm, grapple, reposition, steal, sunder, trip.

## Common AC Modifiers

| Condition | AC |
|---|---|
| Prone (vs melee) | −4 |
| Prone (vs ranged) | +4 |
| Pinned | −4, lose DEX |
| Helpless | −4, lose DEX, melee becomes touch |
| Charging | −2 |
| Fighting defensively | +2 AC, −4 attack |
| Total defense | +4 AC, no attacks |

## Armor Bonus Types

Bonuses of the same type do not stack. Common types: armor (from armor), shield (from shield), natural (natural armor), deflection (rings etc.), dodge (stacks with all others), circumstance.

## Losing DEX to AC

A creature loses its DEX bonus to AC when: flat-footed, unable to react (pinned, helpless, stunned), or when the attacker is invisible.
