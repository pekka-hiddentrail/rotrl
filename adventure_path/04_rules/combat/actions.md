# Actions in Combat

**Triggers:** action, actions, can i, standard action, move action, swift action, free action, full attack, full round, full-round, charge, run, withdraw, five-foot step, 5-foot step, activate, draw, sheathe, cast a spell, casting time

## Action Economy Per Round

| Type | Per Round | Typical Use |
|---|---|---|
| Standard | 1 | Attack, cast a spell, activate item, aid another |
| Move | 1 | Move up to speed, draw/sheathe, stand from prone |
| Swift | 1 | Quickened spells, class abilities |
| Free | Any (GM limits) | Drop item, speak, cease concentration |
| Full-Round | Replaces standard + move | Full attack, charge, run, coup de grace |
| Immediate | 1 (any time) | Off-turn reactions; costs next round's swift action |

**Standard → Move:** a standard action can be used as a move action (not vice versa).

**Full-Round movement:** only a 5-foot step (before, during, or after). Free and swift actions still allowed.

**Restricted activity** (e.g. surprise round): one standard OR one move action only. No full-round actions.

## Miscellaneous

**Use a feat:** feats that grant special combat actions specify their action type (standard, move, swift, etc.) in their description. Passive feats grant bonuses with no action cost.

**Use a skill:** most skill uses are standard actions; some are move actions, full-round actions, free actions, or longer. Check the individual skill description for the action type and whether it provokes AoO.
